# Stories

This folder holds **user stories** for v2 development. Each story is one `.md` file.

- **New story:** Copy `USER_STORY_TEMPLATE.md`, save it here with a descriptive name (e.g. `remove-bud-spud-prompting.md`, `lobby-auto-alias.md`), and fill in the sections.
- **Agent:** When planning an implementation or implementing a story, read the relevant story file(s) in this folder. Use them to scope changes and to ask the developer to test whether the "Done when" criteria are met.
